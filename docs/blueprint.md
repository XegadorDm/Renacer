# **App Name**: Renacer Digital

## Core Features:

- User Authentication: Secure login and registration functionality using email and password.
- Dashboard Navigation: Interactive map of Cauca with clickable regions for case management.
- Case Search: Find existing cases by case number or person's name.
- New Case Form: Create and store new case records including personal details, displacement information, and a personal testimony.
- Case Status Indicator: Visually track the administrative status of each case using a color-coded indicator, with status options: Pending Payment, Pending Collection and No Issue.

## Style Guidelines:

- Primary color: Forest green (#388E3C), symbolizing growth, renewal, and stability for an app focused on recovery.
- Background color: Light beige (#F5F5DC), providing a calming and neutral backdrop to ease the user's experience.
- Accent color: Earthy orange (#D35400), used sparingly to draw attention to key actions and information.
- Font pairing: 'Alegreya', a serif (headlines), paired with 'PT Sans', a sans-serif (body).
- Use clean, easily recognizable icons to represent locations and case statuses, aiding quick identification and navigation.
- Prioritize clear hierarchy and intuitive flow within forms and dashboards to guide users efficiently.
- Use subtle transitions and feedback animations to acknowledge user interactions and maintain engagement.
